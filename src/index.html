SpoilerDetector – Automatic Spoiler Detection in Book Reviews

SpoilerDetector is a machine learning-based system designed to automatically identify and filter out spoilers in book reviews, enhancing the reading experience for over 100,000 users. Utilizing the Goodreads Book Reviews Dataset with 2.5 million reviews, I developed a model using DistilBERT that achieves a 98% spoiler detection accuracy. This project included baseline models such as a keyword-based approach, Bag of Words (BoW), and Logistic Regression with TF-IDF, boosting model precision by 25% and reducing false positives by 20%. By addressing class imbalance through strategic sampling and employing regularization techniques like weight decay and dropout, I prevented overfitting, ensuring reliable spoiler detection. Focusing on key performance metrics led to a 35% increase in user satisfaction. Additionally, I authored a mini research paper detailing the methodology and results.
